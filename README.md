# Python Employee Records CLI

Simple command-line Python program for managing employee salary records.

## Features
- View all employees
- Add employees
- Update salaries
- Delete employees
- Data saved to a text file

All names and data shown in this demo are fictional and used only for demonstration purposes.

## Run the Program

1. Clone the repository

git clone https://github.com/TMGit360/employee-records-manager.git

2. Navigate to the project folder

cd employee-records-manager

3. Run the program

python employee_manager.py

## Requirements

- Python 3 installed
- No external libraries required
